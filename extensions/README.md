# Extensions

This directory contains standalone extension projects that can be installed into the kapsl runtime via `/api/extensions/install`.

## Available Projects

- `s3-rag-extension`: sidecar connector for ingesting RAG documents from Amazon S3.
- `azure-blob-rag-extension`: sidecar connector for ingesting RAG documents from Azure Blob Storage.
- `local-storage-rag-extension`: sidecar connector for ingesting RAG documents from local filesystem storage.
